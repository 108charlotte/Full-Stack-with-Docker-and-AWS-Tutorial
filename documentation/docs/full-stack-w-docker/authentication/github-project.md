# Create a GitHub Repository for your Project
Check out [this tutorial video](https://www.youtube.com/watch?v=-RZ03WHqkaY) for a comprehensive overview of creating a new repository. 

You can name it whatever you'd like (ex. docker-full-stack-tutorial, task-manager-tutorial, etc!), just make sure you remember where to find it. It can be public or private, and once it is created make a local copy on your computer and open it in your editor of choice. [Here is a tutorial](https://www.youtube.com/watch?v=aUhZxPpR5bA) on doing it! 

This tutorial assumes knowledge of React and Django, so if you are unfamiliar with GitHub and programming in general you may want to learn about these frameworks first, before starting this tutorial.
